# Topic 127 Security Policy

This lab demonstrates secure development and industrial cybersecurity controls for an academic EC2 environment.

## Controls
- Recipe integrity checking with SHA-256
- OPC-UA/Modbus process validation
- DevSecOps scans using Bandit, Semgrep, and Trivy
- Audit logs for critical actions
- Incident summary for operational visibility
- Compliance evidence generation

## Limitation
Physical cleanroom sensors and semiconductor equipment are simulated because EC2 does not provide factory hardware.
