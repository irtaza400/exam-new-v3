# Viva Preparation

## Q1: Why simulation?
EC2 has no physical cleanroom sensors or semiconductor machines. The software platform is real; only physical input signals are simulated.

## Q2: Where is cybersecurity?
Recipe integrity, OPC-UA/Modbus validation, DevSecOps scans, audit logs, incident management, and supply chain hash ledger.

## Q3: How does this map to Topic 127?
It covers cleanroom monitoring, ML anomaly detection, process control, supply chain security, worker safety, compliance, and DevSecOps.

## Q4: Can this connect to real factory equipment?
Yes. Replace simulator publishers with real MQTT sensors, OPC-UA endpoints, Modbus PLCs, MES/QMS APIs, and EHS systems.
